iTrade

Intelligent Stock Analytics for Smarter Investing

iTrade is a next-generation stock analytics and portfolio management platform designed to help investors analyze markets, track portfolios, monitor watchlists, and generate AI-driven trading insights.

---

Overview

The goal of iTrade is to provide a scalable and intelligent analytics platform that combines financial data analysis, technical indicators, portfolio tracking, and AI-powered decision support.

The platform is designed using modern software engineering principles including Clean Architecture, Repository Pattern, Service Layer Architecture, JWT Authentication, and Docker-based deployment.
